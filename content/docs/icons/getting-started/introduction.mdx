remocn icons are animated line icons built for video. Each one is a [Lucide](https://lucide.dev) glyph re-authored on the Remotion API, so it draws itself on and then performs a small action — a checkmark strikes, a bell rings, a heart beats — as a pure function of the timeline. Same input frame, same pixels, every render.

Every icon shares one animation contract and one motion system, so switching icons never changes the surrounding scene. Install a single icon with:

```bash
npx shadcn@latest add @remocn/icon-check
```

## The two-phase motion system

An icon runs in two phases:

- **Draw** — the strokes trace themselves on, staggered by path, then a spring settles the whole glyph into place.
- **Action** — a short, icon-specific gesture that reads the meaning at a glance. The action can play once or loop.

The `animation` prop selects which phases run: `draw` stops after the strokes are on, `action` skips the draw and starts mid-gesture, and `both` (the default) draws and then acts.

Compose an icon like any other remocn primitive — offset it in time with `<Sequence>` and place it next to a text animation:

```tsx
import { AbsoluteFill, Sequence } from "remotion";
import { CheckIcon } from "@/components/remocn/icon-check";
import { SoftBlurIn } from "@/components/remocn/soft-blur-in";

export const Scene = () => (
  <AbsoluteFill className="items-center justify-center gap-6">
    <Sequence from={0}>
      <CheckIcon size={64} color="#22c55e" />
    </Sequence>
    <Sequence from={12}>
      <SoftBlurIn text="Done" />
    </Sequence>
  </AbsoluteFill>
);
```

## Shared props

Every icon accepts the same `IconAnimationProps`. This is the single reference — individual icons only add their own defaults, never new controls.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `animation` | `draw` / `action` / `both` | `both` | Which phases run. |
| `loop` | `boolean` | `false` | Repeat the action phase forever. |
| `speed` | `number` | `1` | Timeline multiplier applied before the animation math. |
| `size` | `number` | `48` | Icon width and height in pixels. |
| `color` | `string` | `currentColor` | Stroke color. |
| `strokeWidth` | `number` | `2` | Stroke width in SVG units. |
| `className` | `string` | — | Passed to the root SVG. |

## Loops by default

A few icons express motion that never resolves, so they loop out of the box. Pass `loop={false}` to play them once:

- `loader`
- `refresh-cw`
- `sparkles`
- `flame`
