A shadcn registry for Remotion. You already know the workflow — run `npx shadcn@latest add`, the files land in your project, and they are yours to edit. remocn applies it to video.

Everything is written from scratch against the Remotion API — `useCurrentFrame`, `interpolate`, `spring` — so components seek correctly in the studio and render deterministically. Remotion itself is a prerequisite, not something remocn installs for you.

### New components

The first catalog — text effects like [Typewriter](/docs/typography/typewriter), [Matrix Decode](/docs/typography/matrix-decode), [RGB Glitch Text](/docs/typography/rgb-glitch-text), [Shimmer Sweep](/docs/typography/shimmer-sweep), [Tracking In](/docs/typography/tracking-in), [Staggered Fade Up](/docs/typography/staggered-fade-up), [Slot Machine Roll](/docs/typography/slot-machine-roll), [Inline Highlight](/docs/typography/inline-highlight), [Marker Highlight](/docs/typography/marker-highlight), [Infinite Marquee](/docs/typography/infinite-marquee), and [Perspective Marquee](/docs/typography/perspective-marquee).

Interface blocks — [Terminal Simulator](/docs/ui-blocks/terminal-simulator), [Glass Code Block](/docs/ui-blocks/glass-code-block), [Animated Bar Chart](/docs/ui-blocks/animated-bar-chart), [Animated Line Chart](/docs/ui-blocks/animated-line-chart), [Data Flow Pipes](/docs/ui-blocks/data-flow-pipes), and [Progress Steps](/docs/ui-blocks/progress-steps).

And the first compositions — [Ecosystem Constellation](/docs/compositions/ecosystem-constellation), [Infinite Bento Pan](/docs/compositions/infinite-bento-pan), and [Live Code Compilation](/docs/compositions/live-code-compilation) — full scenes assembled from the primitives above.
