If you build videos with an AI agent, it now knows the catalog. The remocn skill teaches Claude Code and friends which components exist, what each one is for, and how to compose them — so "make me an intro scene" stops being a guessing game.

<ChangelogPreview name="glass-code-walk" />

### New components

- [Glass Code Walk](/docs/ui-blocks/glass-code-walk) — a camera walk across a code block, for showing a diff without narrating it.
- [Terminal Cursor Zoom](/docs/ui-blocks/terminal-cursor-zoom) — the terminal, zoomed to wherever the cursor is working.
- [Logo Enter](/docs/social/logo-enter) — a logo entrance for the end card.

### Improvements

- The [agent skill](/docs/getting-started/agent-skill) ships with install instructions and a component catalog.
- An install-all button on the components page, for anyone who would rather have everything than choose.
